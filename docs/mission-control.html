Interet du projet

Le Track 2 juge deux choses: est-ce que la caption respecte vraiment la video, et est-ce que le ton demande est net. Notre architecture optimise ces deux axes separement pour eviter les captions jolies mais fausses.

Accuracy + Style

Pour le hackathon

Produire un Docker public, robuste, en anglais, sous les limites de temps, avec un JSON strict.

Pour le score

Une seule analyse factuelle solide alimente les quatre styles, ce qui garde la coherence entre captions.

Pour le bonus

Gemma reste dans la couche de style, avec Fireworks documente dans le README et le runbook.

Pour la securite

Pas de cle dans le repo, pas de cache cross-run, pas de hardcoding des clips exemples.

Pipeline interactif

Clique une etape: tu vois ce qu'elle fait, pourquoi elle existe, et quel risque elle reduit.

Operable

01 - Ingest

Le container lit /input/tasks.json au demarrage. Pydantic valide les taches et force les quatre styles obligatoires quand l'entree est incomplete.

Risque reduit: entree invalide, style manquant, score zero sur JSON. Pas de hardcoding Fallbacks partout

Ce qui a ete ameliore

Les modifications sont organisees par impact: robustesse, score, recherche, presentation.

Integre

Validation stricte

Pydantic pour les entrees et sorties, self-check zero cout et contract test.

English only

Filtre runtime et self-check contre les sorties non anglaises ou non ASCII-safe.

Retry reseau

Download stream, retry HTTP, timeouts par tache et fallbacks non vides.

Secrets

Scan literals, .env ignore, aucun credential bake dans Docker.

Model chain

Qwen2.5-VL 7B vers 32B vers MiniMax M3 pour la comprehension.

Gemma style

Gemma 3 27B garde le positionnement bonus pour les rewrites de style.

Direct video

Option Qwen3 Omni/Molmo2 via deploiement dedie, desactivee par defaut.

LoRA ready

Zips audites

Kit, starter v1 et starter v2 inventories. V2 integre dans le canon.

Concurrents

Commandes utiles

Les commandes sont copiables. Les checks sans API et le pont Groq sont utilisables maintenant.

Preflight local 
python scripts/preflight.py
Docker build + run 
python scripts/preflight.py --docker-build --docker-run
Run strict avec API 
set FIREWORKS_API_KEY=...
python scripts/preflight.py --strict --docker-build --docker-run
Run Groq live 
PROVIDER_ORDER=groq,fireworks,openrouter \
MAX_CONCURRENCY=1 NUM_FRAMES=5 FRAME_MAX_EDGE=512 \
INPUT_PATH=data/sample_tasks.json OUTPUT_PATH=out/groq_results_final.json \
python -m app.main
Publier image 
PUBLIC_IMAGE=ghcr.io/<user>/track2-captioner:final make publish
PUBLIC_IMAGE=ghcr.io/<user>/track2-captioner:final make verify-public
